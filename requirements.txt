Software Requirements

Operating System: Windows 10 or above
Programming Language: Python 3.9+
IDE / Editor: Visual Studio Code

Python Libraries

NumPy
Pandas
Scikit-learn
Matplotlib
Statsmodels
TensorFlow / Keras

Data Requirements

Dataset should be in CSV format
The dataset should contain power consumption related attributes
Required features used in this project:
Global Active Power
Global Reactive Power
Voltage
Global Intensity
Dataset should not contain missing or invalid values (handled during preprocessing)

 Functional Requirements

The system should load real-world power consumption data
The system should preprocess data by cleaning and normalization
The system should support multivariate time series input
The system should split data into training and testing sets using time-based splitting
The system should implement a baseline SARIMA model
The system should build an Attention-based LSTM model
The system should train the deep learning model on historical sequences
The system should evaluate performance using MAE and RMSE
The system should save trained models for future use

 Non-Functional Requirements
 
The system should be simple and easy to understand
The code should be modular and well-structured
The system should avoid data leakage
The system should run efficiently on CPU without GPU dependency
The results should be reproducible
